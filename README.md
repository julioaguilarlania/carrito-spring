# Backend para carrito de Compras

## Plataforma
* Java 21
* Spring 6 / Spring Boot 3
* Maven
* H2

## Instrucciones

### Para correr el proyecto
Ejecutar el comando 

    ./mvnw spring-boot:run

o, si tenemos Maven instalado

    mvn spring-boot:run

### Para generar un JAR ejecutable

Correr el comando

    ./mvnw package

o, si tenemos Maven instalado

    mvn package
